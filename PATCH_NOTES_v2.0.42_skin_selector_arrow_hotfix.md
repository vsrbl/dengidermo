# v2.0.43 — skin selector arrow hotfix

Small UI hotfix for the skin selector.

## Fixed

- Skin selector arrows no longer use font glyph positioning.
- Left/right arrows are now CSS-drawn chevrons.
- Both arrows have identical size, baseline, centering and hit area.
- Buttons keep the same width as the fixed skin selector layout.
- Preview remains centered between the arrows.

## Notes

This is intentionally a tiny patch: no gameplay, skin unlock, audio or weapon logic changed.

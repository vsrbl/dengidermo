# PATCH NOTES v2.1.85 — Mirror / Greed / Redline / Lock hotfix

Дата: 2026-07-06

## Исправлено

- `MIRROR PAYOUT` больше не копирует саму себя при выборе boss reward.
- Лимит `MIRROR` теперь восстанавливается после победы над boss/core, а не при переходе на новый loop.
  - Если `MIRROR` копирует boss reward, charge тратится сразу на новый boss-cycle и не восстанавливается мгновенно при следующей комнате.
- `MIRROR PAYOUT` убрана из boss-choice mirror-tag как цель для самокопирования.
- `GOLD FEVER / Золотая лихорадка`: из мобов теперь падает только `GLD`.
  - Обычный `EXP`-дроп с мобов конвертируется в дополнительное золото.
  - `HEA`-дроп с мобов в этой комнате не появляется.
- Casino `LOCK`: если за один spin выпадает несколько новых `LOCK`, фиксируется только первый новый LOCK-слот.
  - Уже зафиксированные старые LOCK-слоты продолжают работать.
  - Повторные новые LOCK в той же прокрутке перебрасываются в обычный не-LOCK символ и не создают новую фиксацию.
- `REDLINE BOOST` получил явный стартовый FX-пакет, экранный kick и звук.
- `REDLINE BOOST` и другие R-активки теперь показывают стак в R-HUD как `x2`, `x3`, если стак больше 1.
- У REDLINE во время действия теперь есть постоянный красный speedline/shell-визуал вокруг игрока.

## Проверено

- `node --check` по `shared`, `src`, `server`, `config.js`.
- Smoke-test casino LOCK: несколько LOCK в одном spin фиксируют только первый слот.
- Smoke-test MIRROR:
  - `sig_mirror_payout` не вызывает `mirror_copy` и не копируется сама;
  - другая boss reward корректно копируется и тратит charge.

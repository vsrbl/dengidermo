# v2.1.150 — process_controller_saw_mass_life_hud_rework

## Process Controller

- SAW переделан из линии-сканера в массовый захват по области курсора.
  - SAW теперь бьёт большим импульсом в точку прицела.
  - Пытается перехватить сразу несколько угроз, пока есть свободные слоты.
  - Имеет более длинную перезарядку, чтобы это было отдельное сильное действие, а не второй CMD.
- Срок контроля теперь индивидуальный у каждого подконтрольного процесса.
  - Новый захват больше не обновляет таймеры уже захваченных процессов.
  - Последний захваченный процесс живёт дольше именно потому, что его таймер только что начался.
  - `CTRL: СРОК +` увеличивает личный таймер каждого нового процесса.
- CTRL-HUD переделан под отдельные маленькие плашки процессов.
  - До 10 компактных ячеек в правом HUD.
  - В каждой ячейке: метка типа процесса, заливка срока контроля, нижняя HP-полоска.
  - Плашки аккуратно отделены от dash/weapon HUD.
- Перенос процессов через портал больше не сбрасывает всем срок до максимума.
  - Без `CTRL: ПЕРЕНОС` процессы очищаются.
  - С `CTRL: ПЕРЕНОС` каждый процесс сохраняет свой оставшийся таймер и безопасно расставляется около игрока.

## Text / balance

- Описания SAW обновлены: теперь это массовый захват, а не линейный разбор.
- Описания `CTRL: СРОК +` обновлены: больше нет текста про обновление срока всем процессам.
- Максимальное количество подконтрольных процессов ограничено 10, чтобы HUD оставался читаемым.

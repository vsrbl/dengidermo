# PATCH NOTES v2.1.84 — Casino Triple-Only / Visual 8BIT Hotfix

Дата: 2026-07-06

## Casino

- Вернул BET terminal к настоящему slot-rule: награда или штраф применяются только если выпали **три одинаковых итоговых символа**.
- Один призовой символ больше не выдаёт GLD/EXP/HP/WPN/ABL/RAR/SKN сам по себе.
- LOCK-логика сохранена:
  - LOCK выпадает в отдельном слоте;
  - растворяется;
  - превращается в случайный символ кроме LOCK;
  - фиксирует только этот слот до закрытия BET terminal;
  - фиксированный символ участвует в следующих проверках на три одинаковых.
- Если LOCK зафиксировал слот, но три одинаковых не собрались, приз не выдаётся.
- Тройной STC теперь даёт static debt; одиночный STC просто часть проигрышной линии.
- Тройной BAD остаётся проигрышем.

## Casino UI / history

- Нижний текст внутри BET terminal снова компактный: только статус + итоговая линия.
- Подробности ставки теперь уходят в feed/history под версией, а не распирают окно казино.
- Убрана прокрутка в нижнем result-блоке казино.
- В feed явно пишется, если не было трёх одинаковых и награда не выдана.

## 8BIT

- Убран фейковый аудио-bitcrusher с общей игровой шины: он ломал звук игры и не мог обработать YouTube iframe.
- Убран отдельный chiptune-mask слой поверх YouTube: он не был настоящей обработкой музыки.
- 8BIT toggle теперь честно работает как visual-only режим для окна YouTube.
- Визуальный фильтр YouTube сделан менее разрушительным.

## Technical

- Версия обновлена до `v2.1.84`.
- `node --check` пройден по `src`, `shared`, `server`, `config.js`.
- Smoke-test `spinCasino()` подтверждает:
  - mixed GLD/BAD/BAD не платит;
  - GLD/GLD/GLD платит;
  - LOCK фиксирует слот без выплаты, пока нет трёх одинаковых;
  - фиксированный слот может собрать triple на следующем спине.

# v2.1.141 — skin_picker_tooltip_cleanup

## Меню

- Убраны всплывающие объяснения с кнопки выбора облика в главном меню.
- Убраны всплывающие объяснения с области выбора облика и кнопок листания облика.
- Убран нативный browser-title с строки статуса облика, чтобы зона выбора не показывала отдельную подсказку при наведении.

## Технически

- `applyStaticI18n()` теперь принудительно очищает `data-explain-*` и `title` у элементов выбора облика, чтобы подсказки не возвращались после смены языка.
- Обновлены версия, build id и cache query до `v2.1.141`.

# PATCH NOTES — v2.1.44 spawn warning dash UI hotfix

## Что изменено

### Зона появления врагов

- Предупреждение появления врагов больше не красное.
- Зона стала серой, спокойной и менее яркой.
- Убран красный текст `INCOMING`, чтобы поздние комнаты не перегружались красными сигналами.
- Красный цвет снова остаётся в основном для урона, опасности и критичных угроз.

### Индикатор рывка

- Убран текст `SHIFT READY`.
- Убран крупный текст `SHIFT 1.2s`.
- Рывок теперь показывается компактными квадратами.
- Восстанавливающийся заряд заполняется внутри маленького квадрата.
- Пока заряд восстанавливается, рядом показывается только маленький таймер без слова `SHIFT`.

## Технически

- Версия обновлена до `v2.1.44`.
- `BUILD_ID`: `spawn_warning_dash_ui_hotfix`.
- Протокол не менялся: `5`.

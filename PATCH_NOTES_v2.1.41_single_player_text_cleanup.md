# PATCH NOTES — v2.1.41 single player text cleanup

## Что изменено

- В русской версии слово `СОЛО` заменено на `ОДИНОЧНАЯ ИГРА`.
- Кнопка запуска без сети теперь называется `ОДИНОЧНАЯ ИГРА`.
- Подсказка кнопки теперь прямо говорит: `Запускает одиночный забег без сети.`
- Сетевые статусы больше не пишут `СОЛО ГОТОВО`:
  - теперь: `ОДИНОЧНАЯ ИГРА ГОТОВА`.
- В английской версии `SOLO` заменено на нормальную формулировку:
  - `SINGLE PLAYER`;
  - `SINGLE PLAYER READY`;
  - `Starts a single-player run without network.`
- В HUD и TAB-панели одиночный режим больше не показывается как `SOLO`-код комнаты; теперь выводится `ОДИНОЧНАЯ ИГРА` / `SINGLE PLAYER`.

## Технически

- Версия обновлена до `v2.1.41`.
- `BUILD_ID`: `single_player_text_cleanup`.
- Механика, сеть, баланс и скины не изменялись.

# v2.1.127 — Living Casino instant actions / BET visual hotfix

## Живое казино

- `BET` теперь срабатывает сразу при выборе в кольце, без второго ЛКМ.
- `COPY` теперь срабатывает сразу при выборе в кольце, без второго ЛКМ.
- Все действия кроме самой `LVC`-пушки больше не становятся выбранным сектором.
- В кольце можно выбрать только `LVC` как пушку для последующего ЛКМ.
- После активации `GUARD`, `CHAIN`, `BET`, `COPY`, `GHOST` выбранной остаётся `LVC`.
- Правая нижняя цветовая LVC-плашка теперь показывает состояние только `LVC`, а не одноразовых действий.

## BET-визуал

- Возвращён предыдущий визуал BET-действия: слот-бёрст и пунктирное квадратное поле на месте активации/результата.
- Убран follow-spin вариант, где слотики постоянно держались над игроком.

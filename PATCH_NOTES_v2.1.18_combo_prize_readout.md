# PATCH NOTES v2.1.18 — combo prize readout

## Зачем
Игроку было недостаточно видно, какой именно приз выдаст комбо. Тип награды был понятен через улучшение/левый блок, но во время боя нужно видеть приз прямо на самом счётчике комбо.

## Изменения
- В верхний счётчик комбо добавлена явная строка приза.
- Теперь комбо показывает текущую ожидаемую выплату прямо над множителем:
  - `ПРИЗ +N GLD`
  - `ПРИЗ +N EXP`
  - `ПРИЗ +N HP`
- В английской версии строка выводится как:
  - `PRIZE +N GLD`
  - `PRIZE +N EXP`
  - `PRIZE +N HP`
- Значение считается из реального текущего состояния:
  - количество убийств;
  - множитель комбо;
  - выбранный тип награды комбо;
  - экономика петли для GLD/EXP;
  - правило HP `убито × множитель × 0.1`, с округлением до целого.
- Визуал оставлен без цветного прямоугольника: это простая верхняя текстовая строка в стиле текущего счётчика.

## Технически
- `VERSION` → `v2.1.18`
- `BUILD_ID` → `combo_prize_readout`
- В снимок комнаты добавлены `prizeType`, `prizeLabel`, `prizeAmount` для `room.combo`.
- `comboSnapshot` теперь получает `players`, чтобы показывать приз согласно выбранному улучшению игрока.
- HUD комбо получил строку `.combo-prize`.

# PATCH NOTES v2.1.100 — Casino Result / Slot Mob Staging Hotfix

Дата: 2026-07-06

## Casino result text

- Убрано смысловое дублирование внутри строки результата BET terminal.
- `ПРОИГРЫШ` больше не дублируется как `ПРОИГРЫШ: -70 GLD` в нижней строке.
- `LOCK-ЯЧЕЙКА` больше не дублируется как `LOCK: ...` в нижней строке.
- Нижняя строка теперь показывает только конкретное последствие:
  - `-70 GLD`
  - `зафиксирован WPN`
  - `НОВОЕ ОРУЖИЕ: ...`
  - `МУТАЦИЯ: ...`
  - `слот сломан, враг собирается`
- Полный результат всё ещё доступен через hover/tooltip.

## Slot mob balance

- Скорость slot mob возвращена к прежнему значению.
- Скорострельность и bullet speed возвращены к прежнему значению.
- HP уменьшено в 4 раза относительно v2.1.99:
  - было `9200`
  - стало `2300`

## Slot mob staged spawn / rebuild

- Первый slot mob больше не появляется сразу визуально после overload.
- Теперь цепочка такая:
  1. в BET terminal проигрывается итог перегрузки;
  2. окно ставки закрывается;
  3. casino chest / сломанный слот ломается в мире;
  4. большие куски и fake GLD/EXP/STC/WPN/ABL частицы физически разлетаются;
  5. частицы слетаются обратно;
  6. только после этого собирается slot mob;
  7. внутри него 3 секунды крутится слот, пока он под защитой;
  8. после выбора режима он становится физически активным.
- Во время `hidden/rebuild/roll` стадий slot mob не получает урон и не перехватывает пули.
- При каждом промежуточном убийстве slot mob снова разваливается на 4 больших куска и много fake-частиц, затем собирается обратно.
- Эффект частиц стал более “весомым”: частицы и куски летят по дугам с псевдо-гравитацией.

## Version

- Версия поднята до `v2.1.100`.

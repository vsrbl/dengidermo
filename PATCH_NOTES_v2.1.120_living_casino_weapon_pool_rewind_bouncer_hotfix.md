# PATCH NOTES v2.1.120 — Living Casino weapon pool / rewind / bouncer hotfix

## Исправлено

- REWIND MARK + Living Casino GHOST больше не рисует линию к `0,0`/за кадр, когда активен ghost-таймер. Rewind tether теперь рисуется только при реальной сохранённой метке.
- Ромбовидный bouncer-моб больше не залипает на месте: если скорость просела или он застрял у стены, ему заново нормализуется направление и скорость.
- Живое казино снова имеет обычный weapon HUD справа снизу.
- Живое казино стартует с пушкой `LVC`, но теперь может открыть обычные `SHG`, `SEK` и `RKT` через WPN-сундуки.
- WPN-сундуки для Живого казино снова могут давать старые оружейные ветки и общие weapon-улучшения, если нужная пушка уже открыта.
- Улучшение стартового damage-сектора переименовано в `LVC` / `LIVE CASINO`, чтобы оно не выглядело как общий DMG-апгрейд.
- Если у Живого казино выбрана не `LVC`, ЛКМ стреляет обычной выбранной пушкой; ПКМ использует secondary этой пушки. Кольцо секторов открывается ПКМ, когда выбрана `LVC`.

## Проверка

- `node --check` пройден по основным JS-файлам.

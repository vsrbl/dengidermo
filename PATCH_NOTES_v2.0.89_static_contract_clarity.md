# v2.0.92 — static/contract clarity pass

- Casino Virus static is now local to the current room and no longer creates next-room Static Storm debt.
- Portals in normal rooms now wait for the clear target and for every live enemy to be dead.
- FAST CLEANUP now means killing every enemy before 42 seconds.
- Contracts no longer spawn on loop 1 and appear often, but not every room.
- Contract cards show the exact future prize preview; ROOM CHECK repeats the exact prize received.
- Contract bonuses show active/pending/used status and clearer descriptions.
- Contract reroll buttons are click-locked and update their displayed uses immediately.
- TAB removes Tape Log, removes the old Run Memory card, and keeps only Contract Chain inside RUN.
- Shifting Walls are now hollow Shifting Zones: slow/light damage areas, not physical walls.
- DMP no longer pulls melee mobs into its field; DMP movement is calmer to reduce jitter.
- VOID CUT line sound cooldown reduced so chained points can each play their click/laser cue.
- More RU/EN naming cleanup: Static Storm, Static Core, Shifting Zones, Blood Payment, Gold Fever, etc.
- Top-right tooltips only open from underlined terminal words.

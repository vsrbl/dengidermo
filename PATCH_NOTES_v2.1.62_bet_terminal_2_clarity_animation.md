# v2.1.62 — BET TERMINAL 2.0 clarity + casino animation

## Что изменилось
- Пересобран интерфейс `BET TERMINAL` под понятный casino-UX, без добавления новой меты.
- В русской версии убран кривой casino-термин из player-facing текста: теперь используются `окна выплат`, `ячейки`, `ставка`, `расчёт`.
- Окно ставок стало стабильным по высоте/ширине: результат, подсказки и кнопки больше не должны дёргать модалку при ставке.
- Добавлена постоянная мини-легенда новых исходов:
  - `HOLD` — следующий сундук получает дополнительный выбор.
  - `LOCK` — следующая ставка получает фиксированный полезный символ.
  - `LINK` — следующая комбо-выплата x2, если не получить урон.
  - `CSH` — безопасный малый cashout.
  - `DEBT` — выплата сейчас, статик-долг потом.
- Кнопки `LOW / MID / HIGH` теперь показывают цену, валюту и риск в стабильной сетке.
- Наведение на ставку теперь объясняет профиль риска, награды и связь с активным контрактом комнаты.
- Добавлена фаза `СТАВКА ПРИНЯТА / РАСЧЁТ ВЫПЛАТЫ / ИТОГ ГОТОВ`.
- Добавлены casino-анимации: бегущая линия расчёта, фиксация ячеек слева направо, разные состояния для выигрыша, проигрыша, долга и jackpot.
- Итог ставки теперь показывает не только цифры, но и объяснение результата: что именно произошло и почему это важно.

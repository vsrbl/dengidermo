# PATCH NOTES — v2.0.44_void_laser_weapon_link_rework

## Главная цель

Переделать два плохо читаемых line-эффекта:

1. `VOID CUT` больше не должен выглядеть как толстая грязная полоса/field.
2. `BULLET CHAIN` у оружия должен читаться как стильная link-связь между врагами, а не как грубый burst/толстая линия.

---

## VOID CUT → VOID LASER

`VOID CUT` оставлен как Q-core, но визуально и механически теперь работает как тонкий лазер из игрока в сторону курсора.

### Новое поведение

- Игрок нажимает Q.
- Из игрока в сторону cursor/aim вылетает длинный тонкий `VOID LASER`.
- Луч остаётся недолго.
- Враги на линии получают урон.
- Enemy bullets на линии стираются.
- Луч больше не выглядит как широкая зона/стена.

### Scaling от уровня core

Апгрейды `VOID CUT` теперь читаемо усиливают:

- длину луча;
- урон луча;
- длительность луча.

Ширина больше не является главным upgrade-направлением. Collision width чуть растёт, но визуал остаётся тонким.

### Визуал

- Новый effect kind: `voidLaser`.
- Тонкое белое ядро.
- Purple/cyan-like void glow.
- Две тонкие dashed side-lines.
- Маленькие квадратные packet sparks вдоль луча.
- Убран старый широкий multi-line `voidLine` вид для этой способности.

---

## WPN chain → WPN LINK

Оружейный chain-эффект переименован в `BULLET LINK +1` в reward cards, чтобы лучше отражать механику.

### Новое поведение

- Попадание оружием связывает врагов тонкой digital-линией.
- Часть урона и elemental-статусы переходят к следующему врагу.
- Каждый стак даёт:
  - `+1` прыжок;
  - больше дальность связи между врагами.

### Scaling дальности

- Ранние стаки связывают только близкие цели.
- Дальняя прокачка постепенно расширяет радиус связи.
- Технический ceiling оставлен высоким, чтобы при долгой прокачке можно было получить огромную цепную сборку.

### Визуал

Добавлены новые fx:

- `weapon_chain_link` — тонкая cyan digital-line между связанными врагами;
- `weapon_chain_lock` — аккуратная квадратная corner-outline рамка вокруг врагов, которые сейчас участвуют в link.

Старый толстый `active_line`/impact-style chain больше не используется для WPN chain.

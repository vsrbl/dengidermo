# v2.1.65 — Procedural Techno Machine Rebuild

## Music
- Replaced the previous active techno/ambient hybrid with a fresh procedural techno machine.
- Added a 16-step state-driven sequencer inspired by drum-machine / game-audio patterns.
- Added synthesized instruments: punchy kick, filtered bass, closed/open hats, clap/noise hit, acid pulse, casino blip, and dark stabs.
- Added separate pattern sets for menu, float, combat, static storm, casino virus, boss, and portal.
- Removed the old drifting melodic/ambient note layer from the active update path.
- Kept air/noise layers very light and above the hum range; the groove now comes from beat and bass, not drone.
- Added controlled fills for boss/storm/casino states.

## Technical
- Added `procedural_techno_machine_v2165` music flavor.
- The new music engine uses no samples, no external tracks, and no bundled libraries.
- Updated build id and version to v2.1.65.

# v2.1.124 — Living Casino HUD fill states hotfix

## Живое казино — HUD выбранного действия

- Убраны числовые секунды из правой нижней плашки выбранного LVC-действия.
- Состояние теперь читается визуальной заливкой:
  - готово — цвет самого действия;
  - активно — серая заливка, которая постепенно пустеет;
  - перезарядка — красная заливка, которая постепенно пустеет;
  - открытое кольцо — светлая рамка выбора.
- Каждому LVC-действию задан свой ready-цвет:
  - LVC — золото;
  - GUARD — циан;
  - CHAIN — фиолетовый;
  - BET — оранжевый;
  - COPY — зелёный;
  - GHOST — синий.
- Ring-плашки также получили отдельный цвет для BET/GHOST, чтобы действия меньше сливались визуально.

# PATCH NOTES v2.1.22 — boss rework: Hunter split + RUSH revisor

## Зачем
После добавления новых боссов часть боёв ощущалась слишком одинаково:
- `HNT / Хор Охотников` давал непонятные красные линии и слишком много обычных врагов.
- `QREV` с копированием активных умений игрока работал нечитаемо и местами не ощущался как нормальный босс.

Этот патч делает два боя проще по смыслу, но разнообразнее по фазам.

## Хор Охотников

### Убрано
- Убрана красная линейная атака Хора.
- Уменьшен призыв обычных врагов.
- Хор больше не заваливает комнату большим количеством подкрепления.

### Добавлено
- После смерти основной оболочки Хор распадается на три малых босса:
  - `HNT-I` — ближний охотник с рывком;
  - `HNT-II` — стрелок;
  - `HNT-III` — ловчий с круговыми метками.
- Портал открывается только после смерти всех малых боссов.
- Основной Хор теперь является первой фазой боя, а не единственным телом босса.

## RUSH / бывший QREV

### Убрано
- Босс больше не копирует активное умение игрока.
- Больше нет случайной активки босса, завязанной на список Q-умений.

### Добавлено
- `QREV` переосмыслен в обычного босса `RUSH`.
- Новый смысл боя:
  - заряжает рывок как красный враг с рывком;
  - рывок быстрее обычного;
  - между рывками стреляет вокруг себя;
  - после рывка даёт круговой залп.
- Визуально использует понятный красный заряд рывка, похожий на уже знакомого врага.

## Техническое
- Добавлены новые типы врагов для малых боссов Хора:
  - `boss_hunter_duelist`
  - `boss_hunter_marksman`
  - `boss_hunter_trapper`
- `PROTOCOL` поднят до `4`, потому что изменился список типов врагов в сетевом снимке.
- Версия обновлена до `v2.1.22`.
- `BUILD_ID = boss_rework_hunter_split_dash_revisor`.

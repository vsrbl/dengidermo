# v2.1.162 — english_menu_text_style_audit

## Что изменено

- Английская версия меню теперь переводит не только кнопки, но и карточки героев:
  - BASE ANTIVIRUS
  - LIVING CASINO
  - PROCESS CONTROLLER
  - описания и подсказки карточек тоже переключаются на EN.
- Сетевой статус меню теперь пересинхронизируется при смене языка:
  - CONNECTING / NETWORK READY / SECTOR NOT FOUND / SECTOR FULL и т.д.
- Начальные заголовки окон INSTALL / WEAPON CHEST / PROTOCOL CHEST / BET TERMINAL обновляются через i18n, а не остаются русскими после переключения языка.
- Английские fallback-лейблы получили дополнительные маппинги, чтобы русские названия героев и служебные выборы не протекали в EN.
- Русские тексты очищены от разговорных формулировок из переписки:
  - `жирные цели` → `цели с большим запасом прочности`;
  - `Казино-пушка` → `казино-модуль`;
  - `враги` в описаниях эффектов заменены на `угрозы`, где это player-facing текст.
- ABL/Protocol wording подчистили в RU/EN: меньше лишних пояснений, больше единого стиля “протокол / модуль движения”.
- Исправлены мелкие формулировки: `ROOM` → `SECTOR` в EN-меню и сетевых ошибках, `ВСЕ ОБЛИКЫ` → `ВСЕ ОБЛИКИ`.

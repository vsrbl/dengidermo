# PATCH NOTES v2.1.94 — Native Cursor Latency Hotfix

## Исправлено

- Видимый кастомный DOM-курсор отключён.
- Игра снова использует нативный браузерный cursor, чтобы убрать микрозадержку и ощущение “вязкости”.
- Наведение по сундукам/install/choice overlay больше не должно давать лага курсора из-за перерисовки DOM-элемента.
- Координаты мыши для прицела/стрельбы по-прежнему читаются через `pointerrawupdate`/`pointermove` и используются напрямую.
- Кнопки и карточки выбора получают обычный pointer-cursor, чтобы было понятно, что они кликабельны.

## Не менялось

- Геймплей сундуков, казино, контрактов и WAGER не трогался.

## Дополнительно

- События мыши больше не обрабатываются тремя путями одновременно.
- Если доступен `pointerrawupdate`, используется только он; иначе `pointermove`, иначе `mousemove`.
- Это убирает шанс, что более старый fallback-event перезапишет свежую позицию курсора/прицела.

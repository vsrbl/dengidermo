# v2.0.36 — fixed skin selector / revealed locked skins

## Skin selector

- Skin selector window now has a fixed width/height so it does not jump when browsing presets.
- Removed long changing note text from the card; the card stays compact and centered.
- Skin name line is fixed and ellipsized instead of resizing/wrapping.
- Locked skins now show:
  - real skin name;
  - rarity;
  - real preview/look;
  - `LOCKED` status.
- Locked skins still cannot be selected until unlocked.

## Visual stick cleanup

- Removed the preview `skin-stick`.
- Removed the small player body aim/barrel tick in world render.
- Cursor/tether and weapon aim logic remain unchanged.

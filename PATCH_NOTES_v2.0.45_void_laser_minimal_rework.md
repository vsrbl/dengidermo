# PATCH NOTES — v2.0.48 void laser minimal rework

- `VOID CUT` сохранён по имени, но по визуалу и поведению это теперь быстрый тонкий `VOID LASER` из игрока.
- Убрано ощущение толстой lingering-полосы.
- Новый VFX: тонкая белая линия с purple signal glow, лёгкий dashed trace и маленькие квадратные digital-маркеры в стилистике игры.
- Луч стартует прямо из игрока, даёт маленький square muzzle и square end mark.
- Длительность сильно короче и ощущается как быстрый лазерный импульс, при этом прокачка всё ещё усиливает длину, урон и длительность.
- Чуть ослаблен slow-компонент, чтобы способность читалась именно как laser strike, а не control-field.

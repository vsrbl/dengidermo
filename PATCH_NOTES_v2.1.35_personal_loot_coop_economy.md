# PATCH NOTES v2.1.35 — personal loot coop economy

## Главное
- Разделена экономика кооператива:
  - общий лут теперь только обычный лут с мобов и лут из BSC-сундука;
  - комбо-призы теперь личные;
  - казино-выплаты теперь личные;
  - лут от Q/способностей теперь личный;
  - траты в сундуках и BET остаются личными.

## Комбо
- Комбо больше не является общей цепочкой команды.
- У каждого игрока теперь своя цепочка комбо.
- Награда комбо начисляется только тому игроку, который сделал это комбо.
- HUD показывает собственное комбо текущего игрока.
- В левом журнале событие выплаты подписывается именем игрока: кто сделал комбо и что получил.

## Лут
- GLD/EXP с обычных мобов остаётся командным: один подобрал — получили все подключённые игроки.
- GLD/EXP из BSC-сундука остаётся командным.
- Казино-лут, Q-казино, ложные выигрыши, карман награды и другие особые источники теперь личные.
- Личные пикапы не смешиваются с командными пикапами.
- Личный пикап с владельцем притягивается только к своему игроку.

## Сундуки и траты
- Траты в сундуках остаются личными: платит только открывающий игрок.
- В левом журнале открытие личного сундука подписывается именем игрока.
- Если сундук платный, в журнале теперь видно, кто заплатил и сколько.
- BSC-сундук может давать общий лут, но сама трата всё равно подписывается открывшим игроком.

## Казино
- BET и его результат подписываются именем игрока в журнале.
- Награды от казино не раздаются всей команде.
- Сигнатура удвоения выплаты больше не удваивает командный лут с мобов для всех; она работает на личных выплатах владельца.

- smoke-тест личного комбо на 2 игрока.
- smoke-тест личного пикапа и командного пикапа.
- unzip -t.

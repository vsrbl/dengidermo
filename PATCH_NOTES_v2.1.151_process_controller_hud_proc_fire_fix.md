# v2.1.151 — process_controller_hud_proc_fire_fix

## Исправлено

- Захваченные стрелковые процессы больше не создают обычный `shot` FX игрока.
  - Убраны лишние звуки/пищание от каждого их выстрела.
  - Муззл/частицы больше не переносятся на тело героя.
- Попадания подконтрольных процессов помечены источником `ctrl_*`, чтобы не играть обычный hit/impact-звук игрока на каждый их выстрел.
- Линия направления у захваченного стрелка больше не крутится сама по себе.
  - В снапшот подконтрольного процесса передаётся реальное `dirX/dirY`.
  - Рендер стрелка использует это направление, а не `now * 2`.

## HUD Контролёра

- Общий CTRL-блок больше не показывает общую длительность процессов.
- Общий CTRL-блок теперь показывает только выбранную команду и её готовность/перезарядку.
- Плашки отдельных процессов вынесены выше основного HUD.
- Плашки процессов стали крупнее и читаемее.
- При отсутствии процессов верхний ряд скрыт, чтобы не поднимать dash без причины.
- Нижние слоты команд для Контролёра визуально стыкуются с CTRL-блоком в один общий контур.
- Добавлен дополнительный отступ для обычных weapon slots, чтобы они не липли к верхним плашкам.

## Дополнительно

- Исправлена видимая версия в верхнем HUD: теперь `v2.1.151`.
- Сохранена логика: Контролёр стартует только с `CMD`; `QRN` и `SAW` открываются через ящик команд.

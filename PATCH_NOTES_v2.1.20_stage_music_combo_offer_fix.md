# v2.1.20 — stage music + combo offer fix

## Музыка комнаты
- Добавлен новый управляющий слой музыки по этапам комнаты: начало, бой, кульминация, очищено/портал открыт.
- Начало комнаты стало тише, медленнее и ниже.
- Кульминация добавляет больше движения, но без резкой верхушки.
- После очистки комнаты музыка уходит в более спокойное разрешение.

## Темы модификаторов
- Музыка теперь немного меняет характер под активный модификатор комнаты: якорь, казино/жадность, статик, призма, охотники, кровь, эхо, движение комнаты.
- Изменения сделаны через регистр, фильтр, ритм и плотность, без отдельных громких мелодий поверх боя.

## Темы боссов
- Крупье Сбоя, Якорный Кассир, Хор Охотников и Q-ревизор получили разные низкие мотивы.
- Боссовые ноты стали заметно тише и ниже, чтобы не давить на ухо.

## Улучшения приза комбо
- В окне улучшений больше не предлагается смена приза комбо на тот тип, который уже стоит.
- В одном окне улучшения может быть максимум один вариант смены приза комбо.
- В одной фазе выбора улучшений один игрок может увидеть смену приза комбо только один раз, и то не гарантированно.

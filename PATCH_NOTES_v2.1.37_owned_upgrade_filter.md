# PATCH NOTES v2.1.37 — Owned Upgrade Filter

## Главное
- Улучшения больше не предлагаются для систем, которых у игрока ещё нет.
- Это касается обычных INSTALL-выборов, WPN-сундуков, RAR/CRS-сундуков и дроновых усилений.

## Дроны
- `DRONE BLAST CHANCE` больше не появляется до получения `DRONE +1`.
- `DRONE ELEMENT LINK` больше не появляется без дрона.
- `DRONE ELEMENT LINK` также больше не появляется, если у игрока ещё нет огня, холода или яда на пулях.

## Оружие
- WPN-сундук больше не показывает серые/запрещённые апгрейды для неоткрытого оружия.
- SHG-усиления доступны только при наличии SHG.
- SEK-усиления доступны только после открытия SEEKER.
- RKT-усиления доступны только после открытия ROCKETGUN.
- Общие оружейные усиления остаются доступными, потому что стартовое оружие уже есть.

## Статусы
- `ELEMENT AMP` и `STATUS SPREAD` больше не появляются до получения хотя бы одного статуса на пулях: огонь, холод или яд.

## Сундуки
- RAR/CRS-сундуки теперь тоже используют фильтр наличия базы.
- Они больше не могут выдать усиление для отсутствующего дрона или отсутствующей статусной ветки.

## Проверки
- `node --check` для `server`, `shared`, `src`.
- Import smoke: `protocol`, `data`, `sim`, `hud`, `audio`.
- Smoke-тест INSTALL без дрона: `DRONE BLAST CHANCE` не выпадает.
- Smoke-тест WPN без SEK/RKT/дрона/статуса: недоступные ветки не показываются.
- `unzip -t`.

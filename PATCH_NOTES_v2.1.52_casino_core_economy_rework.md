# v2.1.52 — Casino Core Economy Rework

## Цель
Первый практический патч казино-ядра: развить уже существующие системы `BET TERMINAL`, `WPN/ABL/RAR/CRS` сундуков, комбо-выплат, контрактов и казино-вируса без добавления чужой меты поверх игры.

## Сундуки и ценность
- Добавлена внутренняя шкала ценности сундуков: `SIMPLE`, `GOOD`, `VALUABLE`, `PREMIUM`.
- `BSC` остаётся простым бесплатным сундуком с GLD/EXP/HEA-дропом.
- `WPN SIMPLE` — обычная цена, 3 выбора оружейных модулей.
- `WPN GOOD/VALUABLE/PREMIUM` — дороже, но даёт больше вариантов и сильнее тянет к реально полезным веткам оружия.
- `ABL SIMPLE` — обычная цена, 3 выбора Q/мобильности.
- `ABL GOOD/VALUABLE/PREMIUM` — дороже, но даёт больше вариантов Q/мутаций/мобильности.
- `RAR` теперь стоит дороже как настоящий редкий сундук; дорогие версии могут давать дополнительную выплату или больше усилений.
- `CRS` остаётся бесплатным опасным сундуком: сильная награда + статик-долг, с повышенной опасностью на дорогих вариантах.
- Обычные комнаты теперь иногда спавнят `GOOD/VALUABLE` WPN/ABL/RAR, а `chill/reward pocket` сохраняет роль дорогого казино-магазина.

## Скейлы стоимости
- Базовые цены обновлены: `WPN 70`, `ABL 65`, `RAR 95`, `CRS 0`.
- Стоимость сундуков теперь учитывает: глубину, тип сундука, `costMul`, ценность сундука и комнатный тип.
- Ставки BET обновлены: `LOW 20`, `MID 55`, `HIGH 135`, с более мягким ростом по глубине, чтобы ставки чаще оставались реальным выбором.

## BET TERMINAL
- Добавлены новые исходы:
  - `HOLD` — следующий WPN/ABL-сундук получает дополнительные варианты выбора.
  - `LOCK` — фиксирует полезный символ для следующей ставки.
  - `LINK` — следующий комбо-платёж удваивается, если игрок не получит урон до выплаты.
  - `CSH` — безопасный cashout, маленькая гарантированная выплата.
  - `DEBT` — выплата сейчас, но усиленный статик-долг потом.
  - `RAR` — редкая награда через BET.
- Ставка у BET в комнате с активным контрактом теперь также считается ставкой на контракт.

## Контракты
- Любая ставка BET в комнате с `SIGNAL CONTRACT` теперь формирует `CONTRACT WAGER`.
- Если контракт выполнен — ставка выплачивает GLD/EXP с множителем серии контрактов.
- Если контракт провален — ставка сгорает и может добавить статик-долг.

## Комбо
- Добавлен `COMBO REEL`: при достаточно сильной серии после выплаты может прокрутиться маленький барабан.
- Возможные исходы: GLD, EXP, HEA, DASH, WPN/HOLD, JCK, иногда STC на высоком риске.
- `LINK` из BET удваивает следующий комбо-платёж, но урон ломает связь.

## UI/подсказки
- Наведение на сундук теперь показывает его ценность: `SIMPLE/GOOD/VALUABLE/PREMIUM`.
- BET-подсказка уточняет, что ставки усиливают контракт комнаты, если он активен.
- Добавлены русские подписи для новых казино-символов и исходов.

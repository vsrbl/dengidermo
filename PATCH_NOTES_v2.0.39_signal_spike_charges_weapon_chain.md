# PATCH NOTES v2.0.39 — signal spike charges + weapon chain

## SIGNAL SPIKE core

- `SIGNAL SPIKE` теперь работает как зарядная точечная способность.
- Уровень core = количество зарядов:
  - I: 1 заряд;
  - II: 2 заряда;
  - III: 3 заряда;
  - дальше каждый апгрейд добавляет ещё +1 заряд.
- Для `SIGNAL SPIKE` снят старый cap в 3 уровня: его можно прокачивать дальше.
- Длительность и урон зоны растут мягко, чтобы сила была в количестве постановок, а не в одной огромной зоне.
- Заряды восстанавливаются по одному; если заряды ещё есть, игрок может ставить следующий spike почти сразу.

## Ability mutation cleanup

- Мутация `CHAIN` убрана из ABL/Ability pool.
- Старые сохранённые `CHAIN`-мутации автоматически вычищаются через `ensureActive`, потому что id больше нет в `ACTIVE_MUTATIONS`.
- Идея цепочки перенесена туда, где она читается лучше: в оружейные модификаторы.

## New WPN upgrade: BULLET CHAIN +1

- Новый WPN chest upgrade: `BULLET CHAIN +1`.
- Попадание оружием перебрасывает часть урона на ближайшего врага.
- Каждый стак добавляет +1 прыжок цепи.
- Цепь переносит часть elemental-эффектов: fire/freeze/poison.
- Верхний лимит технически ограничен 100 прыжками, чтобы очень долгая прокачка могла превращать это в массовую цепную сборку без бесконечных циклов.
- Визуал цепи: короткие cyan digital линии между врагами + label `WPN CHAIN xN`.

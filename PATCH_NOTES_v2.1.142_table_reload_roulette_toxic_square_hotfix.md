# v2.1.142 — table_reload_roulette_toxic_square_hotfix

## Исправлено

- TABLE / карта-ловушка Живого казино теперь имеет отдельную перезарядку, синхронизированную с сектором в кольце.
- Перезарядка BET и TABLE теперь тикает в общем состоянии Живого казино, а не только в момент обработки ЛКМ/LVC.
- После срабатывания TABLE сектор кратко помечается активным и затем уходит в перезарядку, чтобы действие нельзя было спамить через кольцо.

## Баланс

- Базовый урон RLT / Рулетки снижен в 2 раза: `78 → 39`.
- Базовая дальность RLT сокращена примерно в 1.5 раза:
  - время жизни `2.25 → 1.50`;
  - максимальная дистанция `980 → 653`.

## Визуал

- Токсичные снаряды RLT больше не получают хвосты, боковые планки или отростки от общего элементального оверлея.
- Для RLT все элементальные эффекты теперь остаются квадратными: простая внутренняя/внешняя квадратная подсветка без лишних выступов.

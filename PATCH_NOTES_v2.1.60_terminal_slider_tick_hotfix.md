# PATCH NOTES v2.1.60 — TERMINAL SLIDER TICK HOTFIX

## Исправлено
- Заменён звук ползунков громкости: вместо мягкого нисходящего `boop` теперь короткий сухой терминальный щелчок.
- Убрано падение высоты тона, из-за которого звук воспринимался как “булькающий”.
- Увеличен интервал между превью-звуками при перетаскивании ползунка, чтобы он не тарахтел слишком часто.
- Превью ползунков остаётся в общем SFX/UI-пуле: если убавить SFX, щелчок ползунков тоже станет тише.
- Добавлен маленький фильтрованный relay/contact transient без режущих высоких частот.

# PATCH NOTES v2.1.25 — ten loop finale + run summary

## Зачем
Игра получила явную цель прохождения: дойти до конца 10 циклов, победить финального босса и завершить забег через финальный портал.

## Цель забега
- Цель теперь считается как 10 циклов.
- Один цикл = 4 комнаты: GRID / VOID / CORE / BOSS.
- Всего цель прохождения: 40 комнат.
- Босс в конце 10-го цикла считается финальным.
- В HUD добавлена строка цели: `ЦЕЛЬ X/10 ЦИКЛОВ` / `GOAL X/10 LOOPS`.

## Финальный портал
- После победы над финальным боссом портал больше не ведёт в следующую комнату.
- Вход в финальный портал завершает прохождение.
- Фаза забега меняется на `won`, симуляция больше не продолжает обычный бой/выбор улучшений.
- Финальный портал сохраняет итоговую статистику и отправляет событие `run_complete`.

## Итоговая таблица забега
После завершения появляется отдельный экран итогов:
- циклы;
- комнаты;
- убийства;
- побеждённые боссы;
- GLD;
- EXP;
- HEA;
- полученный урон;
- лучшее комбо;
- лучшая серия комнат без урона;
- лучшая серия быстрых комнат;
- контракты;
- сигнатуры угрозы;
- данные по игрокам;
- оружие;
- Q;
- спутники, рывки, дроны, орбитали, скин.

## Финальная музыка
- Добавлена отдельная длинная тема завершения.
- Она включается только после полного прохождения.
- Это не боевой слой: музыка спокойнее, приятнее и сидит в низком/среднем регистре.
- Тема продолжает играть, пока открыт экран итогов.

## Финальный босс
- Финальным боссом назначается отдельный бой в конце 10-го цикла.
- Сейчас используется CROUPIER / Крупье Сбоя как финальный узел системы.
- Финальный босс получает больше здоровья, чтобы ощущаться как конец маршрута.

# v2.1.3 — Crowd Forms Director

- Added internal crowd-form spawn grammar for enemy packs.
- The director now shapes packs as WALL / FAN / RING / NEST / LANE / STREAM / PINCH / CLOUD depending on room archetype, active modifier and loop.
- Late loops can combine two forms in one wave, e.g. WALL+LANE, RING+NEST, FAN+STREAM, PINCH+WALL.
- Hunter Waves and Casino Virus mob events now also use shaped spawns.
- No new combat labels are shown around mobs; the system changes spawn geometry and pressure only.

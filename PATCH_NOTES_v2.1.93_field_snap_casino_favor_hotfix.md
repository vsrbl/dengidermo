# PATCH NOTES v2.1.93 — Field Snap / Casino Feed / Contract Favor Hotfix

Дата: 2026-07-06

## Fixes

- `FIELD SNAP / СТЯЖКА ПОЛЯ` больше не запускает экранный sweep-line через весь экран при активации.
  - Оставлен локальный импульс/поле и лёгкий kick.
  - Убрана лишняя большая горизонтальная линия, которая выглядела как чужой visual.

- Casino history/feed очищен от лишних slot-debug деталей.
  - В историю больше не пишутся строки вида `S1`, `S2`, `S3` для каждого слота.
  - История пишет компактный итог: ставка, есть ли `x3`, что выплачено, LOCK-фиксация, STC/пустой итог.
  - LOCK summary теперь выглядит как `WPN` или `WPN x2`, без `S1->WPN`.

- Контрактные призы теперь группируются по одинаковому типу.
  - Два одинаковых `ПЕРЕБРОС ВЫБОРА` отображаются как один приз с `2 заряда`, а не две одинаковые плашки по `1 заряд`.
  - Страховка/очистка тоже отображают количество зарядов, если накопились.

- Исправлено исчезновение оставшегося reroll-заряда на следующем сундуке.
  - UI больше не вычитает локальный расход поверх уже уменьшенного серверного остатка.
  - Если было 2 заряда и один использован, следующий WPN/ABL/boss choice снова видит оставшийся заряд.

## Tech checks

- `node --check` по основным JS-модулям.
- ESM import smoke-test для `shared/sim.v2-1.js` и `src/hud.v2-1.js`.
- Casino LOCK smoke-test: lock summary больше не содержит `S1/S2/S3`.
